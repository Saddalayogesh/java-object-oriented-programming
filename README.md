# Java Object-Oriented Programming

This repository contains Java programs created to practice and understand the core concepts of Object-Oriented Programming (OOP). It serves as a learning resource for beginners who want to strengthen their Java fundamentals through hands-on coding.

## 📌 About

The project demonstrates the implementation of various OOP concepts using Java. Each program focuses on a specific concept and provides practical examples to improve problem-solving and coding skills.

## 🚀 Concepts Covered

- Classes and Objects
- Constructors
- Instance and Local Variables
- Methods
- Encapsulation
- Inheritance
- Method Overloading
- Method Overriding
- Polymorphism
- Abstraction
- Interfaces
- Packages
- Access Modifiers
- `this` and `super` Keywords
- Static Members
- Object Creation and Initialization
